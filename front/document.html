Pulse Pilot Documentation

How to use this application?

To start off, you made it here already! Very impressive. To start the application you want to press on the green button on your dashboard. This will start the application and you will be able to see the data on the dashboard. If you want to stop the application, you can press the red button on the dashboard. If you want to see the data from the previous sessions, you can press on the "Your Sessions" button on the sidebar. If you want to log out, you can press on the "Log Out" button on the sidebar. If you want to shutdown the RaspberryPi, you can press on the "Shutdown RaspberryPI" button on the sidebar.

Why did I make this application?

I made this because I love working out. Thing with working out is that I love to visualize my progress and know that what I'm doing is safe I can see all my previous sessions and see how I've improved over time. I can also see if I'm pushing myself too hard and if I need to take a break. You can see that on the app but the machine will start vibrating and making noise if you go in the danger zone. I can also see if I'm not pushing myself hard enough and if I need to push myself more.
